## 2. 캠페인 목표 & 핵심 메시지

- **광고 목적:** 인지(Awareness) — 신규 브랜드 '모레'의 이름과 "가볍게 챙기는 단백질"이라는 인상 각인
- **핵심 메시지(한 문장):** "바쁜 아침, 모레 한 팩이면 오늘의 나를 가볍게 채운다."
- **타겟 인사이트:** "챙겨야 하는 건 알지만 아침엔 시간도 입맛도 없다" → 부담 없는 한 모금으로 죄책감 해소
- **CTA:** 브랜드명 + 로고 + 슬로건 노출 (마지막 3초)

## 3. 브랜드 로고

- 워드마크 "MORE" + 잎 심벌 + "PLANT PROTEIN" 태그라인

## 7. 프롬프트 수정 전/후 기록

### 사례 ① — 씬2 전환 컷 (톤·일관성)

**수정 전 (A):**
```
냉장고 안의 단백질 음료, 제품 사진, 밝고, 컬러풀하게
```
결과: 채도 높은 마트 진열대 느낌, 제품 형태가 생성마다 바뀜, 브랜드 톤(세이지/미니멀)과 불일치.

**수정 후 (B):**
```
냉장고 내부 시점에서 손을 뻗어 종이팩을 집으려는 모습. 세이지 그린 무광 종이팩(지붕형, 흰색 원형 뚜껑, 무지 라벨). 부드러운 아침 자연광, 크림 톤, 은은한 필름 그레인, 시네마틱 광고 스타일, 가로 구도 1536×1024
```
결과: 채도 낮은 아침 톤으로 통일, 팩 형태 안정적으로 고정, 손 동작으로 '전환' 서사 강화.

**결과 변화 & 원인 분석(A→B):**
- **변화:** 고채도 스튜디오 제품샷 → 저채도 아침 광고 톤 + 팩 형태 고정.
- **원인:** A는 "밝고, 컬러풀하게" 같은 **톤을 통제하지 않는 형용사** + 형태 정보 부족 → 모델이 학습 분포의 '전형적 제품 사진'(고채도)으로 수렴. B는 광원·색·질감·형태를 구체 명사로 못박아 **모델의 자유도를 좁혀** 브랜드 톤으로 유도한 것이 직접 원인.

### 사례 ② — 제품 형태 (유리병 → 종이팩)

**수정 전:**
```
세이지 그린 유리병(크림색 미니멀 라벨, 무광 마감), 스튜디오 부드러운 조명, 제품 히어로 샷
```
결과: 화장품 세럼병처럼 보임. "유리병+미니멀 라벨+스튜디오 조명"이 스킨케어 광고 문법과 겹침.

**수정 후:**
```
우유팩·두유팩 같은 종이 카톤(게이블탑) 형태, 세이지 그린 무광 종이 질감, 흰색 원형 뚜껑, 무지 라벨, 아침 식탁 위 자연광
```
결과: 식물성 음료 카테고리가 즉시 연상됨. 화장품 느낌 제거.

**원인:** 용기의 '소재·형태'가 카테고리 인식을 좌우. 유리병+스튜디오 조명 = 화장품, 종이팩(테트라/게이블탑) = 유제품·식물성 음료로 모델이 해석. 형태를 "종이 카톤"으로 명시하자 원하는 카테고리로 정확히 이동.

| 비디오 | Sora | 시네마틱 질감 | 이미지 입력 미지원(T2V만) → 키비주얼을 텍스트로 재현. 과한 모션 시 왜곡 → "최소한의 움직임" 명시 |
| 오디오 | OpenAI TTS-1 | 톤 일관된 한국어 음성 | 음악 생성 불가 → 효과음+내레이션 구성 |

### 8-2. 선택 우선순위 (품질·제어·속도·비용)

학습·과제 성격상 **비용(크레딧)과 제어(일관성)를 최우선**, 품질은 "720p로 충분", 속도는 후순위.

1. **비용(1순위):** 이미지로 구도 확정 후에만 영상화. Sora·720p, 씬당 4초, 6→4→3씬 축소.
2. **제어(2순위):** 제품·톤 묘사 프롬프트 고정, 로고 SVG 직접 제작, 씬4 이미지 처리로 완전 통제.
3. **품질(3순위):** 과제 허용 스펙(720p)에서 타협. 1080p·상위 모델은 옵션 병기.
4. **속도(4순위):** 대기열은 대체 도구(Veo 3.1)로 우회.

### 8-3. Text-to-Image vs Image-to-Video vs Text-to-Video

| 방식 | 특징 | 본 프로젝트 적용 |
|------|------|------------------|
| T2I | 텍스트→정지 이미지. 세밀한 통제, 저비용 재생성 | GPT Image 2로 채택, 키비주얼 확정 |
| I2V | 확정 이미지를 첫 프레임으로 모션 부여. 일관성 최상 | 당초 계획했으나 Sora 이미지 입력 미지원으로 불가 |
| T2V | 텍스트만으로 영상. 자유도 높음, 이미지 100% 일치 어려움 | 차선책 채택, 키비주얼을 텍스트로 재현 |

**선택 기준:** 이상적으론 I2V(일관성 최상)를 원했으나 도구 제약으로 불가 → **T2I로 목표 프레임 확정 + T2V로 재현**하는 하이브리드로 일관성과 실현 가능성을 동시 확보.

## 9. 일관성 제어 & 통합 보정

### 9-1. 캐릭터·스타일 일관성 제어 (파라미터 기능이 없는 도구에서)

GPT Image 2 / Sora는 `--cref`·`--sref`·ControlNet 같은 파라미터형 레퍼런스가 없으므로 3계층으로 통제:

1. **프롬프트 고정:** 공통 스타일 문구 + 제품 상세 묘사를 모든 프롬프트에 동일 문장으로 반복 (어휘 단위 스타일 시드).
2. **키비주얼 기준화:** T2I로 '목표 프레임' 확정 후 T2V가 이를 텍스트로 재현 (이미지가 사실상 레퍼런스).
3. **후처리 통제:** 로고는 SVG로 100% 동일 재사용, 씬4는 이미지 합성으로 브랜드 요소 고정, CapCut 색·밝기 보정.

### 9-2. 도구 간 결과물 통합 보정 (불일치 해소 절차)

| 불일치 유형 | 보정 기준 & 절차 |
|-------------|------------------|
| 해상도·비율 | 이미지 1536×1024(3:2) ≠ 영상 1280×720(16:9). 기준을 **영상 규격**으로 통일 → CapCut에서 이미지 16:9 크롭(피사체 중앙 배치로 손실 최소) |
| 색감·톤 | 기준색 = 씬 키비주얼의 세이지/크림 톤. 씬별 밝기·채도 차이를 CapCut 색보정으로 정합 |
| 프레임레이트 | 최종 24~30fps로 통일 |
| 오디오 레벨 | TTS 보이스오버 기준, 효과음은 배경 레벨로 낮춤 |
| 컷 전환 | 씬 간 짧은 디졸브(약 0.3초)로 톤 차이 완충 |

## 11. 심층 문항 대응 (확장 시나리오)

### 11-1. 같은 스토리보드를 다른 도구로 만든다면?

| 교체 도구 | 가장 달라질 점 |
|-----------|----------------|
| 이미지: Midjourney | `--sref/--cref`로 스타일·제품 고정이 더 강력 → 일관성이 프롬프트 반복보다 안정적. 대신 사실적 제품 질감은 GPT Image 2가 유리 |
| 비디오: Veo 3.1 / Kling | 이미지 입력(I2V) 지원 시 키비주얼을 첫 프레임으로 직접 사용 → 이미지-영상 일치도 급상승. 지금의 "텍스트 재현" 손실 사라짐. 모션 자연스러움·인물 왜곡도 도구별 차이 큼 |
| 오디오: ElevenLabs / Suno | ElevenLabs는 음성 감정·톤 제어 세밀. Suno 추가 시 BGM 확보로 톤앤매너 폭 확대 |

**핵심 예측:** 가장 크게 바뀌는 건 **비디오 단계**. I2V 지원 도구로 바꾸면 "이미지→영상" 일관성 문제가 근본 해결되어, 지금의 하이브리드 절충(T2I+T2V)이 불필요해진다.

### 11-2. 크레딧이 더 부족해진다면?

- **씬 구성:** 3씬 → 2씬(제품 발견 + 브랜드 CTA)으로 추가 압축. 서사보다 제품·슬로건 인지 집중.
- **도구 선택:** 영상 생성을 줄이고 이미지 여러 장 + CapCut 줌·패닝(Ken Burns)으로 **정지 이미지 기반 모션 광고**로 전환.
- **편집 전략:** 컷 수를 줄이고 각 컷을 길게, 슬로건 자막·보이스오버로 메시지 밀도 유지.

### 11-3. 60초 버전을 15초로 줄여야 한다면?

| 구간 | 처리 |
|------|------|
| 유지 | 제품 등장 핵심 컷(발견) + 만족 반응 1컷 + 브랜드 CTA. "누가·무엇을·왜" 최소 골격 |
| 삭제·압축 | 분위기·라이프스타일 부연 컷, 반복 앵글, 도입 빌드업 삭제. 여러 컷 감정을 대표 1컷으로 압축 |
| 메시지 재구성 | 내레이션을 슬로건 한 줄로 축약, 텍스트는 마지막 CTA에 집중. 60초 '설명형' → 15초 '각인형' 전환 |

**원칙:** 길이가 줄수록 **서사(기승전결)보다 브랜드·제품 인지**를 우선. 15초는 "기억에 남는 한 장면 + 슬로건"이 60초 완결 스토리보다 효과적. — 본 과제(10초)에서 씬1(도입)을 뺀 것과 동일한 판단 논리.
