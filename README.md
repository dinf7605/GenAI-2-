# 모레 MORE — AI 기반 브랜드 광고 패키지

식물성 단백질 음료 브랜드 **모레(MORE)**의 10초 이내 광고 제작 과제 결과물입니다.
기획 의도 → 프롬프트 → 결과물 → 통합 편집으로 이어지는 멀티모달 AI 제작 파이프라인을 직접 설계·구현했습니다.

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 브랜드 | 모레(MORE) — 무가당 식물성 단백질 음료 (가상 브랜드) |
| 타겟 | 25~35세 도심 직장인 |
| 캠페인 목적 | 인지(Awareness) |
| 핵심 메시지 | "바쁜 아침, 모레 한 팩이면 오늘의 나를 가볍게 채운다." |
| 슬로건 | "내일의 나를 위한 한 모금" |
| 영상 길이 | 약 8.5초 (3씬 구성) |
| 영상 스펙 | 1280×720 / 720p, MP4 (H.264/AAC) |

---

## 서사 구조 (메시지 전달형)

제품 발견(전환) → 한 모금의 만족(해결) → 브랜드 인지(CTA)

> 당초 4씬(씬1=아침의 분주함/문제 제시 포함)으로 설계했으나, 실제 제작에서 씬1을 제외하고 3씬으로 완성했습니다.
> 도입 컷 없이 제품 발견부터 시작해도 메시지가 충분히 전달되고, 크레딧·길이 효율이 높기 때문입니다.

---

## 사용 도구

| 유형 | 도구 | 용도 |
|------|------|------|
| 이미지 | GPT Image 2 | 씬별 키비주얼(목표 프레임) 및 씬4 최종 소스 |
| 비디오 | Sora (Text→Video) | 키비주얼을 텍스트로 묘사해 영상 생성 |
| 오디오 | OpenAI TTS-1 | 슬로건 보이스오버 |
| 로고 | SVG 직접 제작 | 워드마크(글자 선명도 확보) |
| 편집 | CapCut | 컷 편집·자막·오디오·합성 |

## 폴더 구조

```
more_ad_package/
├── 01_storyboard/      스토리보드(기획) 문서 PDF
├── 02_keyvisuals/      씬별 키비주얼 이미지 (GPT Image 2)
├── 03_logo/            브랜드 로고 (밝은/어두운 배경용)
├── 04_final_cta/       로고·슬로건 합성한 최종 CTA 이미지
└── 05_video/           최종 광고 영상 (별도 업로드)
```

---

## 제작 파이프라인

1. **기획** — 브랜드/캠페인/핵심 메시지 정의
2. **이미지** — GPT Image 2로 씬별 키비주얼 확정 (구도·톤·제품 고정)
3. **비디오** — 키비주얼을 텍스트로 정밀 묘사해 Sora로 생성 (씬당 4초)
4. **오디오** — OpenAI TTS-1로 슬로건 보이스오버
5. **편집** — CapCut에서 트리밍·합치기·로고·자막·음성 합성

자세한 씬별 프롬프트·의사결정 기록은 `01_storyboard`의 PDF를 참고하세요.

---

## 제작 중 주요 의사결정

- **제품 형태**: 초기 유리병 → 화장품처럼 보여 종이팩(게이블탑)으로 전환
- **로고**: 이미지 AI는 글자가 뭉개져 SVG로 직접 제작 후 합성
- **비디오**: 도구가 이미지 입력을 지원하지 않아 img2video → text2video로 전환, 키비주얼을 텍스트로 재현
- **씬4**: 로고·자막 선명도를 위해 영상 대신 정지 이미지로 처리 (최종 영상에 이미지 소스 포함)
- **크레딧 절약**: 6씬 구상 → 4씬 설계 → 실제 3씬, Sora·720p 선택
